# Code of Conduct

## Our Standard

Be respectful.  
Don't be an asshole.  
Be cool, calm, and constructive like a zen master.

## Expected Behavior

- Treat contributors with respect, even in disagreement.
- Give technical feedback with evidence and without personal attacks.
- Assume good intent and ask clarifying questions before escalating conflict.
- Keep collaboration focused on improving the code and the project.

## Unacceptable Behavior

- Harassment, insults, demeaning language, or discrimination.
- Repeated hostile behavior after being asked to stop.
- Personal attacks in issues, pull requests, reviews, or chat.

## Enforcement

Project maintainers may remove comments, close threads, or restrict participation
for behavior that violates this code of conduct.
